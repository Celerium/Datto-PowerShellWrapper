# Remove-DattoBaseURI

## SYNOPSIS
Removes the Datto base URI global variable.

## SYNTAX

```powershell
Remove-DattoBaseURI [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
The Remove-DattoBaseURI cmdlet removes the Datto base URI global variable.

## EXAMPLES

### EXAMPLE 1
```powershell
Remove-DattoBaseURI
```

Removes the Datto base URI global variable.
