# Get-DattoBaseURI

## SYNOPSIS
Shows the Datto base URI global variable.

## SYNTAX

```powershell
Get-DattoBaseURI [<CommonParameters>]
```

## DESCRIPTION
The Get-DattoBaseURI cmdlet shows the Datto base URI global variable value.
